# KPI's Segurança do Trabalho — Pessoas e Processos

Dashboard estático para a US Vale Verde, com duas visões integradas às planilhas públicas de KPIs do Google Sheets.

O navegador consulta diretamente as abas por Google Visualization Query (GViz). Para refletir alterações, basta editar as planilhas e recarregar o dashboard. Nenhuma cópia local dos dados é mantida.
